Airline Database

Overview

This project was based of off my CGS 2545C database concepts class. We practiced using SQL and created a mock database of airline employees, destinations, various flights, and more. This was a great way to implement skills learned from the course and expressed it through that project.

Lessons Learned

Designed and developed a relational database to mimic airline operations data, including employee records, flight schedules, payroll, and location data. Utilized MySQL to create normalized schemas, optimize query performance, and ensure data integrity. Implemented SQL queries for reporting and mimicked payroll of airline employees.
